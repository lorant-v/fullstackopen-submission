# fullstackopen-submission

This repository stores exercise submissions for the Full Stack Open course: https://fullstackopen.com

Recommended repository layout:

<pre>
part0
part1
  courseinfo
  unicafe
  anecdotes
part2
  phonebook
  countries
</pre>

When adding each project, include the full project folder but omit `node_modules`. Keep each part's exercises grouped under the `partN/` folders.
